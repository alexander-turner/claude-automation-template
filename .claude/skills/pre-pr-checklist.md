# Pre-PR Checklist

Before pushing or creating a PR, ensure you have completed these steps:

## Required

- [ ] **Self-critique completed** - Launched a critique sub-agent to review the diff
- [ ] **Critique feedback addressed** - Fixed all legitimate issues found
- [ ] **Validation passed** - lint and typecheck succeeded (see output above)

## Reminders

- Follow the full PR creation workflow in `.claude/skills/pr-creation.md`
- Include a clear summary of changes in the PR description
- Add the Claude session URL to the PR body
